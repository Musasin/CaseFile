◇ 起動方法 ◇
  Mac OS : 結月ゆかりの事件簿(Mac版)フォルダを任意の場所にコピーし、その後フォルダ内のCaseFile.appから起動してください。
  Windows: 結月ゆかりの事件簿(Win版)フォルダを任意の場所にコピーし、その後フォルダ内のCaseFile.exeから起動してください。

◇ 削除方法 ◇
	インストールを行う方式では無いのでそのままフォルダごと削除して下さい。

◇ 操作方法 ◇
	基本的に操作マウス操作で進行します。
	画面上をクリック: ページ送り
	NOTEボタンをクリック: 物語が進むとメモ書きが増え、館内図などが確認できるようになります。
	ITEMボタンをクリック: 獲得したアイテムやメモが確認できます。
	SKIPボタンをクリック: 次の選択肢までのスキップが行えます。
	LOGボタンをクリック: バックログの確認が行えます。
	画面右上の◀︎ボタンをクリック: 設定ウィンドウを呼び出し、文字表示速度や音のON/OFF、フルスクリーンかどうかなどが切り替えられます。
	右クリック: 特定のウィンドウを表示中の場合にはそのウィンドウが閉じられ、何も表示していない場合にはアイテムウィンドウが開かれます。
	CTRLキー: 押している間早送り再生を行います。
	ALTキー: オートモードの切り替えを行います。
	マウスホイール: バックログの呼び出しと、バックログのスクロールが行えます。
  その他画面説明を同フォルダのジャケット裏.pngに記載
